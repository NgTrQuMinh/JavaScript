I. Giới thiệu ngôn ngữ JavaScript

1. Khái niệm

JavaScript (JS) là ngôn ngữ lập trình kịch bản nhẹ, được sử dụng phổ biến nhất để tạo ra các trang web tương tác. Nó là một phần của bộ ba công nghệ web chuẩn: HTML, CSS và JavaScript.

2. Vai trò

  • Tương tác người dùng: Xử lý sự kiện click, nhập liệu, hiệu ứng cuộn trang...
  • Xử lý dữ liệu: Tính toán, kiểm tra tính hợp lệ của form.
  • Giao tiếp Server: Gửi và nhận dữ liệu mà không cần tải lại trang (AJAX).
  • Backend: Với Node.js, JS có thể xây dựng ứng dụng phía máy chủ.

3. Cài đặt môi trường

Bạn chỉ cần:

  • Editor: Visual Studio Code (Khuyên dùng).
  • Runtime: Trình duyệt web (Chrome, Firefox) hoặc Node.js để chạy trên máy tính.

4. Hướng dẫn chạy JavaScript

Có 3 cách chính để thêm JS vào dự án:

// 1. Inline: Viết trực tiếp trong thuộc tính HTML
<button onclick="alert('Hello!')">Click me</button>

// 2. Internal: Trong thẻ <script> của file HTML
<script>
    console.log("Chạy JS nội bộ");
</script>

// 3. External: Tạo file .js riêng (Khuyên dùng)
<script src="script.js"></script>

II. Cú pháp cơ bản của JavaScript

1. Quy tắc câu lệnh

  • Kết thúc mỗi câu lệnh bằng dấu chấm phẩy (;).
  • Phân biệt chữ hoa chữ thường: Variable khác với variable.

2. Lệnh Comment

// Đây là ghi chú 1 dòng

/* Đây là ghi chú
   nhiều dòng
*/

3. Biến và Khai báo (var, let, const)

Từ khóa Mô tả Phạm vi (Scope)
var Cách cũ, có thể khai báo lại. Global / Function
let Có thể thay đổi giá trị, không thể khai báo lại. Block {}
const Hằng số, không thể thay đổi giá trị. Block {}

4. Quy tắc đặt tên biến

  • Bắt đầu bằng chữ cái, dấu gạch dưới (_) hoặc $.
  • Không bắt đầu bằng số.
  • Nên dùng quy tắc camelCase (ví dụ: userName, studentAge).

5. Kiểu dữ liệu

  • Number: 1, 3.14
  • String: "Hello", 'JS'
  • Boolean: true, false
  • Null: Giá trị rỗng có chủ đích.
  • Undefined: Biến chưa được gán giá trị.
  • Object: { name: "An", age: 20 }

6. Kiểm tra kiểu dữ liệu

Sử dụng toán tử typeof:

let a = 10;
console.log(typeof a); // "number"

let b = "JavaScript";
console.log(typeof b); // "string"
